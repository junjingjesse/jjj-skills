# 故事创作工具

## 快速开始

当用户触发此技能时，直接说：

**好的！告诉我你想写什么类型的故事，我来帮你！**

然后等待用户描述他们的写作需求（题材、困惑、具体问题等）。

---

## 用户需求识别

根据用户的问题，快速定位需要的方法论：

| 用户需求 | 推荐模块 |
|---------|---------|
| 不知道怎么开头、情节怎么组织 | 情节构建 |
| 人物单薄、不立体、没特点 | 人物塑造 |
| 场景干巴巴、不生动 | 场景描写 |
| 文字平淡、没风格 | 文体技巧 |
| 写完了不知道改什么 | 修订指南 |
| 全部都要学 | 完整工作流 |

## 第一模块：情节构建

### 核心概念

**Plot vs Story**
- Story（故事）：按时间顺序"然后...然后..."
- Plot（情节）：因果关联"因为...所以..."
- 好的情节 = 每一个情节点都导致下一个情节点

### Freytag金字塔（七要素）

```
         Climax（高潮）
            /\
           /  \
          /    \
  Rising Action     Falling Action
  （上升情节）       （下降情节）
        /              \
       /                \
      /                  \
Exposition          Resolution
（背景/开端）          （解决）
      \                /
       \              /
        \            /
         Inciting Incident
           （导火线/触发事件）
```

| 要素 | 说明 | 示例 |
|-----|------|------|
| Exposition | 背景介绍 | 介绍主角、环境、前史 |
| Inciting Incident | 导火线 | 打破平衡的事件，主角目标出现 |
| Rising Action | 上升情节 | 冲突加剧，障碍增多 |
| Climax | 高潮 | 最高冲突点，主角与阻力最大对决 |
| Falling Action | 下降情节 | 高潮余波，冲突收束 |
| Resolution | 解决 | 主要冲突的最终解决 |
| Denouement | 结局 | 解释剩余悬念，完整收尾 |

### ABDCE五幕结构

| 字母 | 含义 | 说明 |
|-----|------|------|
| A | Action（动作） | 开篇必须有可见行动，角色被迫进入剧情 |
| B | Background（背景） | 解释角色为何行动及其动机 |
| D | Development（发展） | 主角应对障碍的系列过程 |
| C | Climax（高潮） | 戏剧性爆发点，角色做关键选择 |
| E | Ending（结局） | 角色与故事状态的最终变化 |

### 场景五要素

1. **行动**：角色做什么
2. **对话**：角色说什么
3. **细节**：感官细节（视觉、听觉、嗅觉等）
4. **内在视角**：角色的内心感受
5. **明确的开始与结束**：场景有清晰的起止点

### 三种场景类型

- **争执场景**：双方有不可调和的矛盾
- **诱惑场景**：一方试图改变另一方
- **谈判场景**：双方有共同目标但利益冲突

### 故事Pitch公式

一句话概括故事：
- **触发事件** + **角色** + **冲突/障碍** + **目标任务**
- 例：当___发生，___必须克服___才能___

### React/Reflect/Reveal

- **React**：角色即时反应（身体或语言）
- **Reflect**：内心权衡利弊
- **Reveal**：向读者揭示角色过往或深层个性

## 第二模块：人物塑造

### 角色构思两要素

**1. 身体（Physical）**
- 从头到脚的外形：身高、脸部细节、肤色、体型、姿态
- 即使是外星生物也要有具体身体特征

**2. 声音（Voice）**
- 说话方式、口音、语速
- 折射出地域背景、个性、情绪

### 愿望与目标

- 每个角色必须有**想要的东西**（外在目标）
- 同时要有**内心渴望**（内在需求）
- 欲望驱动行动，行动推动情节

### 角色缺陷

- **无缺陷 = 无趣**：完美角色无法引发冲突
- 缺陷例子：想保持身材却想吃甜甜圈，想当好人又想发财
- 缺陷与欲望交织：角色想得到某物，但缺陷使过程艰难

### 视角选择

| 视角 | 特点 | 适用场景 |
|-----|------|---------|
| 第一人称 | 强烈代入感，局限性 | 内心独白多 |
| 限制性第三人称 | 从单一角色角度，保留内心独白 | 现代小说常用 |
| 全知视角 | 上帝视角，洞悉所有角色内心 | 需要展现多角色 |
| 多视角 | 丰富信息，复杂结构 | 长篇小说 |
| 第二人称 | 亲密感/讽刺感 | 实验性文学 |

### 跨距离写作

- 写与自己差异大的角色（年龄、性别、背景）可获得更多客观视角
- 需要大量观察、接触与研究，保持共情
- 远离"典型"与"刻板"，让角色有内在独特性与矛盾
- 旁观者视角（如Nick Carraway）可带来更客观的视野

### 打破第四面墙

- 直接对读者发问，增强共鸣
- 如Harriet Beecher Stowe在《汤姆叔叔的小屋》中的写法

### 怪癖与细节

- 小动作、口头禅、穿着偏好
- 具体细节 > 笼统形容
- "明星般帅气" → "头发乱糟糟，右耳有助听器"

### 对话的力量

- 对话不是"报告信息"，而是"行动"
- 每句对话都应该有角色间的"交换"或"博弈"
- 让人物通过语言"呈现"内心，而非旁白解释

### 内在冲突

- 角色必须有自我矛盾
- 即使是"反派"也有自我合理性
- 用具体细节表达"大主题"（爱情、种族、阶级）

## 第三模块：场景描写

### 显著细节原则

- **不是所有细节都重要**：只选能推动情节或刻画角色的细节
- 细节要"多功能"：同时完成多重任务

### 冰山理论

- 只呈现关键表面
- 隐藏的信息让读者参与构建
- 读者能"感觉"到水面下的八分之七

### 五感描写

- **视觉**：颜色、光线、形状
- **听觉**：声音、音调、节奏
- **嗅觉**：气味、香味、腐味
- **味觉**：味道、温度、质感
- **触觉**：材质、温度、重量

### 细节选择原则

- 从角色视角出发：角色性格决定他们关注什么细节
- 初入高级餐厅的女孩 → 注意烛台、银器、桌布质感
- 长期在寒冷中的人 → 见到冻伤可能并不震惊
- 细节必须服务角色或情节，否则宁缺毋滥

### 避免刻板印象

- 对陌生文化或时代：先研究，再动笔
- "不偷懒"：仔细研究与深度思考，而非依赖刻板印象
- 跨语言描写：母语表达情感更真实，但需有叙事动机

### 角色的"主观能动性"

- 女性角色必须有"自我目标"和"成长历程"，不是剧情附属品
- 任何角色都需要有"行动决策、思考与追求"
- 角色因欲望而行动，因缺陷而产生冲突

### 抽象与具体的平衡

- 先用具体细节"呈现"主题，再恰当使用少量抽象概括
- "Show, don't tell"并非只用细节堆砌，需选择最能传达主题的细节
- "Show to tell"：通过精巧场景暗示，再配合简练评点

### 结尾的"飞跃"

- 有效结尾在最后时刻让作品产生回响
- 升华或改变前文提供的信息
- 让读者重返前文时产生新理解

### 写作声音的寻找

- 通过记日记或随手笔记捕捉灵感
- 像调音一样反复推敲，直到找到"真声音"
- 保持个人独特声线

- 写自己熟悉/关心的世界
- 借角色关注的细节，展现人物个性并推动剧情
- 场景可成为"角色"：多角色对同一地点不同看法

- **场景(Scene)**：实时呈现，包含对话、动作、细节，让读者身临其境
- **概述(Summary)**：快速跳过大段时间或背景，避免冗长
- 两者交替使用，保持叙事节奏

### 写作练习：限制性写作

- 带限制的练习（如只写十秒、只写外在动作）可打破惯性思维
- 聚焦具体描述，跳过抽象或内心描写

### 三种研究

1. **功能性研究**：查询历史、地理、名称等实用信息，确保细节准确
2. **启发性研究**：接触艺术、音乐、阅读，激发灵感
3. **想象性研究**：在脑海中"观看"场景，让故事世界"充满活体感"

### 说服力

- 说服力 = 让读者忘记这是虚构
- "坏小说"：读者能看见"木偶线头"，感到故事僵硬
- 好的描述让文字"唤起五感"，使虚构比真实更真实

### 展现vs告知

- **Show, don't tell**：通过行动、对话、细节让读者自行感受
- 不要直接陈述情感，要让读者自己体会

### 场景暗示角色心境

- 时间、季节、空间要素暗示角色当下的心境或情节转折
- 季节/假期气氛与家庭聚会交错，角色彷徨、思念等情绪呼之欲出
- 平凡事件（如买菜、布置晚餐）只要与角色情感结合，也能成为关键片段

### "偷听者"视角

- 偷听者获取他人隐私言语，通常引入新张力或伏笔
- 旁观者视角让读者觉察更多未明说的真相或情感痕迹

### 儿童式观察

- 刻意维持对事物的陌生感，看似愚笨
- 忽略固有认知，像孩子注意到事物表面的纹理、声音或材质
- 编辑阶段再理性筛选信息

### 跨文化写作

- 若写陌生国家/文化，避免依赖刻板印象
- 通过研究和想象获取准确、生动的细节
- 只需少量"精准且用心"的细节让读者信服

### 风格的调整

- 词汇选择可唤起特定时代/地域氛围
- 语态（主动vs被动）影响表达效果
- 个人语言"滤镜"可通过有意识练习调整

## 第四模块：文体技巧

### 主动语态

- 主动语态 > 被动语态
- "门被推开" → "他推开们"

### 简洁原则

- 每句话都应该：加深角色 或 推进情节
- 第二稿比第一稿精简约10%
- 删除"不属于大卫雕像的那部分大理石"

### 语言的"真"

- 写"真切"的句子
- 放下逃避心理，把角色真实的想法展现给读者
- 读者对"人性深层"和"现实压力感"产生共鸣

### 朗读检验

- 文字与口语相连
- 朗读能发现"想要跳过的段落"——那些对读者也无益处

### 风格的建立

- 从熟悉的素材出发
- 融入想象力与文学加工
- "真中带假、假中有真"

## 第五模块：修订指南

### 21条修订清单

1. 拼写与小错误
2. 删除不必要的词（精简）
3. 避免替读者思考（减少"告知"）
4. 删除舞台调度、繁琐背景
5. 角色动机一致
6. 开篇动作：第一个句子就要有行动
7. 故事连贯性（物理及情感细节一致）
8. 场景完整性（五要素检查）
9. 每章节以活跃方式开始
10. 使用主动语态
11. 场景与环境有效结合
12. 角色行为可信度
13. 段落/章节过渡流畅
14. 五幕式结构检查（ABDCE）
15. 展现风险（角色有冒险，故事有张力）
16. 后果与代价
17. 每句是否加深角色/推进情节
18. 第二稿精简10%
19. 舍弃无救的内容
20. "寻找你真正想表达的"
21. **读者不再感到困惑**

### 困惑 ≠ 悬念

- **困惑**：读者"看不懂、搞不清"（坏）
- **悬念**：deliberate的剧情设置（好）
- 读者能清晰理解情节，才算达标

### 大胆删除

- 删减原则：保留服务剧情与主题的文字
- 若删除后节奏更好、可读性提升，则可舍弃
- 四个版本最终只留一个是常态

## 激发问题（对话中使用）

### 情节相关
- "这个情节点的'因为...所以...'是什么？"
- "如果没有这个事件，故事还能往下走吗？"
- "高潮是全书最紧张的时刻吗？"

### 人物相关
- "这个角色最想要什么？"
- "他的缺陷是什么？"
- "反派为什么不觉得自己是坏人？"
- "这句话是角色在说，还是你在说？"

### 场景相关
- "读者能'看见'这个场景吗？"
- "这个细节同时服务哪几个目的？"
- "如果删掉这段描写，故事会受影响吗？"

### 修订相关
- "这句话是'展示'还是'告知'？"
- "读者读到这段会困惑吗？"
- "删掉这整段，作品会更好吗？"

### 原生景观（Primal Landscape）

- 童年/早期生活常驻地是对写作影响最深的地方
- 是情感、记忆和人生观的不竭灵感源
- 自己熟悉的地方有"内线优势"，对人文、隐秘细节了如指掌

### 类型文学（Genre）

- 常见类型：犯罪、奇幻、恐怖、推理、科幻、浪漫
- Noir（黑色小说）：聚焦人性阴暗面、命运无常、诱惑与堕落
- 核心：不是特定地点年代，而是幽暗人性、危险欲念
- 写作建议：避免套用陈旧比喻，用当代语言创造新的黑暗世界

### 写作心态

- "写作理想"而非"写作规则"：每位作家可发展独特风格
- "转动飞轮"：信任自我、专注投入，持续磨砺
- "刀尖向前"：全力投入写作信念

### 反馈技巧

- 给反馈原则：真诚而善意、注重作品独特有趣的地方
- 反馈是给作者的"致信"，帮助作者看得更透彻
- 接收反馈：帮助更清晰看待自己作品，可能为长篇项目提供启发

## 写作流程参考

### 新手写作顺序
1. **构思**：想一个"一句话故事"（目标+障碍）
2. **人物**：设计2-3个主要角色（愿望+缺陷）
3. **结构**：用ABDCE画大纲
4. **场景**：列出需要的关键场景
5. **初稿**：按场景写，不要追求完美
6. **修订**：按21条清单逐条检查

### 进阶技巧
- 写不出来时：换一个"进屋方式"（视角、顺序、结构）
- 角色"拒绝"按你的方向走时：说明角色已"生长"，重写情节
- 模仿喜欢的作品：拆解它的结构，为我所用

## 推荐写作原则

1. **动作开场**：第一个句子就要有行动
2. **欲望驱动**：角色必须"想要"什么
3. **缺陷让角色活着**：完美角色无趣
4. **场景即舞台**：Nothing happens nowhere
5. **每句双重功能**：加深角色 或 推进情节
6. **删除是朋友**：第二稿比第一稿少10%
7. **读出来检验**：跳过的段落，读者也会跳过
