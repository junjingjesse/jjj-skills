# SEO竞品分析工具

## 快速开始

用户触发此技能后，直接说：

**请提供你要分析的关键词（可多个，用逗号或换行分隔）**

然后等待用户输入关键词。

---

## 方法论

### 竞争判断逻辑

| 指标 | 说明 | 机会 |
|------|------|------|
| 首页内页比 | 首页多=竞争大，内页多=机会大 | 内页多更有利 |
| 大站数量 | 大站多=难竞争 | 少大站更容易 |
| allintitle数 | 专门优化少=机会大 | 数量少更有利 |
| 域名可注册性 | 有可注册域名=机会 | 可注册=机会 |

### 搜索意图类型

| 类型 | 说明 | 商业价值 |
|------|------|----------|
| INFO | 信息查询 | 低 |
| NAV | 导航寻找 | 中 |
| TRANSAC | 交易购买 | 高 |
| COMM | 交流讨论 | 低 |

### KGR vs 竞品分析
- **KGR技能**：粗筛，找KGR低的词
- **本技能**：细查，逐个分析具体对手

## 执行流程

### 第1步：接收关键词

用户输入一个或多个关键词（从KGR技能输出的值得做关键词中选择）。

**支持批量**：多个关键词用逗号、换行或空格分隔。

### 第2步：批量Google搜索分析

**重要**：每个关键词搜索后**不要关闭页面**，保持打开状态，方便用户后续查看。

对每个关键词执行Google搜索：
- 提取Top 10结果的页面类型（首页/内页）
- 记录域名分布（大站/小站/个人）
- 提取搜索结果中的**People Also Ask**相关内容，辅助判断搜索意图

**搜索意图判断方法**：
- 看AI Overview的摘要
- 看People Also Ask的问题类型
- 看搜索结果的页面类型（折扣页=TRANSAC，文章页=INFO，商品页=TRANSAC）

### 第3步：allintitle分析

搜索 `allintitle:关键词` 看有多少页面专门优化这个关键词。

### 第4步：可视化输出

**重要**：分析完成后**不要关闭任何Chrome tab**，保持搜索结果页面打开，让用户可以后续查看。


## 评估标准

| 指标 | 低竞争 | 中竞争 | 高竞争 |

| allintitle | <1000 | 1000-10000 | >10000 |
