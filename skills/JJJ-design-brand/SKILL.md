# JJJ-design-brand 品牌设计技能

## 技能定位

这是品牌设计的**综合应用技能**，学会后可以为虚构或真实公司设计完整的品牌视觉系统。

---

## 核心框架

品牌设计流程：
1. **发明客户** → 确定品牌概念
2. **命名** → 品牌名称
3. **视觉研究** → 找灵感
4. **标志设计** → 核心视觉
5. **色彩与字体** → 视觉系统
6. **品牌指南** → 文档输出

---

## 技能模块

### 模块一：发明你的客户 (Invent Your Client)

这是品牌设计的起点——先创造一个虚构客户，给创意最大的自由空间。

#### 为什么要用虚构客户？
- 现实客户有既定框架，限制创意
- 虚构客户可以天马行空
- **创意最大化**

#### 如何发明客户？

**步骤1：定义品牌理念**
- 这个公司卖什么/做什么？
- 目标人群是谁？
- 核心理念是什么？

**步骤2：从形容词出发**
- 从描述品牌的形容词开始
- 例如：有机、高科技、复古、手工、豪华

**步骤3：构建公司历史**
- 公司成立于何时？
- 创始人的背景故事？
- 使命是什么？

#### 示例
```
公司概念：火星旅游公司
→ 形容词：未来、冒险、探索
→ 成立于：2088年
→ 使命：让普通人也能探索太空
```

### 模块二：命名 (Naming)

名称是品牌的第一印象。

#### 命名方法

**方法1：思维导图法**
1. 列出品牌相关的关键词
2. 以每个词为中心向外发散
3. 筛选出有潜力的词

**方法2：词根拆分**
- 将单词拆解成不同部分
- 重新组合成新词

**方法3：典故引用**
- 从历史、文化中找词源
- 让名称有故事感

#### 好的品牌名特征
- 简洁易记
- 发音流畅
- 有意义关联
- 域名可用

#### 示例
```
品牌概念：蜗牛食品车
→ escargot（法语蜗牛）
→ S-CAR-GO（车+去）
→ Revolution（革命）
→ Snail Trail（蜗牛轨迹）
→ 选择：S-CAR-GO
```

### 模块三：视觉研究 (Visual Research)

在动手设计前，先做调研。

#### 视觉研究的目的
- 找到设计方向
- 避免抄袭
- 获得灵感

#### 研究方向

**1. 历史研究**
- 过去的相关设计
- 设计史上的经典案例
- 行业演变

**2. 当代研究**
- 竞争对手的设计
- 行业趋势
- 新兴风格

**3. 跨领域研究**
- 不同行业的视觉语言
- 音乐、电影、艺术

#### 研究技巧
- 用 Pinterest 建立图库
- 用颜色分类图片
- 记录直觉喜欢的部分
- **不是复制，而是启发**

### 模块四：标志设计 (Logo Design)

标志是品牌的核心视觉。

#### 标志类型

| 类型 | 特点 | 示例 |
|------|------|------|
| 文字标志 | 用品牌名作为logo | IBM, Google |
| 图标标志 | 用图形代表品牌 | Apple, Nike |
| 组合标志 | 图形+文字 | 多数品牌 |

#### 标志设计流程

**步骤1：分析字母形态**
- 公司名中的每个字母分析
- 找有特点的字母（a, g, o）
- 探索不同字体的形态

**步骤2：大小写实验**
- 全大写：正式、稳重
- 全小写：友好、现代
- 首字母大写：平衡

**步骤3：简化**
- 用最少的线条表达
- 从复杂到简化
- 测试可辨识性

**步骤4：测试不同应用**
- 小尺寸下是否清晰
- 单色下是否有效
- 不同背景下的效果

#### 字��设计技巧
- **a, g**：变化丰富，适合变形
- **o**：简洁最好，不要过度复杂
- **R, G**：有动态感，适合品牌

### 模块五：色彩 (Color)

色彩是品牌情绪的直接表达。

#### 色彩三属性

| 属性 | 定义 | 影响 |
|------|------|------|
| 色相 | 颜色的基本色 | 情感基调 |
| 明度 | 颜色的明暗 | 层次感 |
| 饱和度 | 颜色的鲜艳度 | 活力程度 |

#### 色彩情绪

| 颜色 | 情绪 | 适用 |
|------|------|------|
| 红色 | 活力、激情、警示 | 餐饮、运动 |
| 蓝色 | 信任、专业、平静 | 科技、金融 |
| 绿色 | 自然、健康、生长 | 环保、健康 |
| 黄色 | 快乐、温暖、能量 | 零售、娱乐 |
| 黑色 | 高端、神秘、正式 | 奢侈、时尚 |
| 白色 | 简洁、干净、纯洁 | 科技、医疗 |

#### 品牌色彩搭配

**主色 + 辅色 + 强调色**
- 主色：60%（品牌主调）
- 辅色：30%（支持）
- 强调色：10%（突出重点）

### 模块六：字体 (Typography)

字体是品牌的"声音"。

#### 字体选择原则
- 与品牌调性匹配
- 区分标题与正文
- 不超过 2-3 种字体

#### 字体类型

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 衬线体 | 传统、优雅 | 奢华品牌 |
| 无衬线体 | 现代、简洁 | 科技品牌 |
| 手写体 | 友好、创意 | 餐饮、创意 |
| 黑体 | 力量、权威 | 运动品牌 |

#### 定制字体
- 选择现有字体后调整
- 修改特定字母
- 创造独特字形

---

### 模块七：品牌应用 (Brand Applications)

品牌在各种场景中的应用。

#### 常见应用场景
- 名片
- 信纸
- 包装
- T恤
- 社交媒体头像
- 网站 banner

#### 应用原则
- **一致性**：保持视觉统一
- **适配性**：不同场景调整
- **灵活性**：留有弹性空间

---

### 模块八：品牌指南 (Brand Guidelines)

品牌指南是品牌的"宪法"。

#### 品牌指南内容

```
1. 品牌故事
   → 公司理念
   → 品牌愿景

2. 标志使用
   → 标志版本（横版、竖版）
   → 安全空间
   → 最小尺寸
   → 禁止使用

3. 色彩系统
   → 主色/辅色/强调色
   → 色值（CMYK, RGB, HEX）

4. 字体系统
   → 标题字体
   → 正文字体
   → 使用规范

5. 图形元素
   → 辅助图形
   → 图标风格

6. 应用示例
   → 名片
   → 包装
   → 数字应用
```

#### 品牌指南的形式
- PDF（数字版）
- 印刷版（实体手册）

## 设计流程

### 标准流程（6步）

```
1. 发明客户
   → 定义品牌概念
   → 设定客户画像

2. 命名
   → 头脑风暴
   → 筛选名称

3. 视觉研究
   → 收集参考
   → 找方向

4. 标志设计
   → 草图
   → 优化
   → 定稿

5. 色彩与字体
   → 选择色彩
   → 搭配字体

6. 品牌指南
   → 整理文档
   → 展示应用
```

## 常见问题

### Q：没有设计基础怎么做品牌？
- 先从文字标志开始
- 用现有字体
- 颜色选简单的

### Q：如何判断设计好不好？
- **可辨识**：缩小后能认出
- **简洁**：不过于复杂
- **一致**：各元素协调

### Q：品牌色怎么选？
- 从行业惯例出发
- 从产品特性出发
- 不要超过 3 种颜色

## 补充模块：品牌设计的深度技巧

### 补充一：伪造你的历史

为虚构公司编写历史，提供设计背景和情感共鸣。

**关键问题**：
- 公司什么时候成立？
- 谁是创始人？
- 公司的哲学理念或态度？
- 公司的目标是什么？
- 品牌背后有怎样的故事？

**写作风格**：根据品牌调性调整（幽默/严肃/正式）

### 补充二：拆开你的形容词

通过形容词定义品牌，然后转化为视觉语言。

**步骤1：选择形容词**
- 例如：gourmet（精致）、French（法式）、Libertarian（革命性）

**步骤2：展开为视觉**
- gourmet → 精致、优雅风格
- French → 文化、知识性、时尚感
- Libertarian → 革命性、抗议元素

### 补充三：字体形态探索

字母形态分析与变形是标志设计的关键。

**关键字母**：a、g、o 具有丰富的变化性

**探索方法**：
- 大小写转换产生不同效果
- 修改标点符号改变感觉
- 单个字母形态分析
- 字母之间的互动与节奏

### 补充四：精细调整与迭代

通过微小调整逐步优化标志。

**调整方法**：
- 延长字母臂部
- 调整横杆位置与角度
- 加入象征性元素
- 每次调整后进行对比分析

### 补充五：色彩策略

**选择方法**：
- 通过情感联想（如自然→绿色）
- 通过品牌形容词
- 通过视觉研究提取

**注意事项**：
- 测试正反色对比
- 考虑印刷与屏幕差异（CMYK vs RGB）
- Pantone色用于印刷

### 补充六：辅助字体选择

**辅助字体作用**：补充主标志字体，传递不同层次信息

**选择原则**：
- 与主字体相辅相成
- 覆盖标题/正文/副标题
- 建立字体套件

### 补充七：标志与图标设计

**设计原则**：
- 简化几何形态
- 考虑黑白效果
- 迭代与细节雕琢
- 赋予多重含义

**动态表现**：现代标志需要考虑动画效果

### 补充八：视觉研究与Mood Board

**研究方法**：
- 历史研究：从过去设计获取灵感
- 当代研究：竞争对手与行业趋势
- 跨领域研究：不同行业的视觉语言

**Mood Board构建**：
- 收集历史与当代图像
- 筛选并编辑图像
- 整合到品牌发展指南

### 补充九：品牌发展指南

品牌指南应包含：
1. 品牌故事与历史
2. 标志使用规范
3. 色彩系统
4. 字体系统
5. 图形元素
6. 应用示例

**动态构建**：品牌指南应随着设计迭代不断完善
