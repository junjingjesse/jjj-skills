# JJJ-design-basics 平面设计基础技能

> 基于 CalArts 平面设计基础课程压缩（63个核心文件）

## 技能定位

这是平面设计的**底层技能**，学会后可以独立完成海报、传单、封面等基础设计。

---

## 核心框架：设计的四大元素

设计由四大元素构成：
1. **图像 (Image)** - 照片或插图
2. **文字 (Type)** - 排版与字体
3. **形状 (Shape)** - 几何图形
4. **颜色 (Color)** - 色彩搭配

将这四个元素**组合**起来，就是设计。

## 模块一：图像制作 (Image Making)

### 1.1 图像的类型

| 类型 | 定义 | 示例 |
|------|------|------|
| **定指图像** | 精确表现物体本身，无附加含义 | 苹果的图像就是苹果 |
| **暗示图像** | 赋予象征意义，传达深层信息 | 苹果+蛇=诱惑 |

### 1.2 图像制作技巧

**核心原则：**
- **过程 > 结果**：创作的核心是"生成"，不要被完美束缚
- **从简单开始**：用基础形状（圆形、方形、三角形）就能传达含义
- **实验精神**：尝试不同媒介和方式，打破常规

**技巧分类：**

1. **线条与工具**
   - 不同工具产生不同线条质地
   - 紧凑线条 = 严谨；松散线条 = 自由

2. **体积与色彩**
   - 不需要线条，色彩本身可以表达体积
   - 通过明暗、阴影创造立体感

3. **裁剪与可识别性**
   - 裁剪距离决定图像的可识别程度
   - 极近距离或极远距离都可以是有效选择

4. **低技术创作**
   - 扫描仪、复印机可以产生意外效果
   - 盲画、连续线条是很好的实验方法

### 1.3 图像制作流程

```
1. 选择物体 → 确定要表达的物体
2. 实验媒介 → 尝试不同工具和材料
3. 迭代优化 → 不追求一次完美，多次改进
4. 意义赋予 → 考虑定指还是暗示表达
```

## 模块二：排版 Typography

排版是**文字的视觉安排**，核心是让文字既好读又好看。

### 2.1 基础术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 基线 | Baseline | 字母排列的参考线 |
| x高度 | X-height | 小写字母的高度 |
| 升部 | Ascender | 超过x高度的字母部分(b,d) |
| 降部 | Descender | 低于基线的字母部分(p,y) |
| 字距 | Kerning | 单个字母之间的间距 |
| 追踪 | Tracking | 整体文字块的间距 |
| 行距 | Leading | 行与行之间的垂直距离 |

### 2.2 字体点数系统

- **1点 = 1/72英寸**
- 常见字号：
  - 标题：72-120pt
  - 子标题：24-36pt
  - 正文：9pt
  - 脚注：6-7pt

### 2.3 字体分类

**Serif 字体（有衬线）：**

| 分类 | 特点 | 示例 |
|------|------|------|
| Old Style | 粗细对比小，斜线结构 | Plantin |
| Transitional | 粗细对比增大 | Times New Roman |
| Modern | 粗细对比极大，垂直 | Bodoni |
| Egyptian | 衬线与笔画同粗 | Clarendon |

**Sans Serif 字体（无衬线）：**

| 分类 | 特点 | 示例 |
|------|------|------|
| Grotesque | 结构简单，略带不规则 | Arial |
| Geometric | 几何形状构建 | Futura |
| Humanist | 带手写特征 | Gill Sans |

### 2.4 对齐方式

| 方式 | 特点 | 适用场景 |
|------|------|----------|
| 左对齐 | 左侧对齐，右侧自然 | 正文、阅读 |
| 两端对齐 | 左右都对齐 | 杂志、报纸 |
| 居中 | 居中排列 | 标题、邀请函 |
| 右对齐 | 右侧对�� | 特殊排版 |

### 2.5 行距规则

- **标准行距**：字体大小的 1.2-1.5 倍
- **推荐**：9pt 字体配 16pt 行距
- **最佳行长**：60-70 字符（约9-12个单词）

### 2.6 字体的功能性与表现性

- **功能性**：可读性、清晰度
- **表现性**：传达情感和风格
- 选择字体时需要平衡二者

## 模块三：形状与图形 (Shape)

### 3.1 形状的方向性与情感

| 形状 | 方向性 | 传达情感 |
|------|--------|----------|
| 圆形 | 均衡、无方向 | 和谐、柔和、流动 |
| 方形 | 水平和垂直 | 稳定、可靠、坚固 |
| 三角形 | 对角线方向 | 动态、紧张、锐利 |

### 3.2 标记、图标与符号

| 类型 | 定义 | 示例 |
|------|------|------|
| 标记 | 只有外形，无内涵 | 圆形、方形 |
| 图标 | 具象表达，易于理解 | 垃圾桶=删除 |
| 符号 | 抽象概念，需学习 | 红十字=医疗 |

### 3.3 图形-背景关系

- **正空间**：画面中的主体
- **负空间**：背景和空白
- 大元素显得更近，小元素显得更远
- 高对比度=前景感，低对比度=背景感

## 模块四：颜色 (Color)

### 4.1 颜色的三大属性

| 属性 | 定义 | 影响 |
|------|------|------|
| 色相 | 颜色的基本种类 | 情感基调 |
| 明度 | 颜色的明暗程度 | 层次感 |
| 饱和度 | 颜色的鲜艳程度 | 活力程度 |

### 4.2 颜色轮

- **相邻色**：色轮上相邻的颜色，和谐搭配
- **互补色**：色轮上相对的颜色，强烈对比

### 4.3 颜色模式

| 模式 | 原理 | 适用 |
|------|------|------|
| RGB | 加法（光） | 屏幕显示 |
| CMYK | 减法（油墨） | 印刷 |
| Spot Color | 专用单色 | 特殊颜色 |

### 4.4 颜色情感

| 颜色 | 情绪 |
|------|------|
| 红色 | 活力、热情、警示 |
| 蓝色 | 信任、稳定、专业 |
| 绿色 | 自然、健康、生长 |
| 黄色 | 快乐、温暖、能量 |
| 黑色 | 高端、神秘、正式 |
| 白色 | 简洁、干净、纯洁 |

## 模块五：构图 (Composition)

### 5.1 视觉对比的六大类别

| 对比类型 | 定义 |
|----------|------|
| 形状 | 不同形状之间的对比 |
| 规模 | 大小元素之间的对比 |
| 重量 | 明暗色调带来的重量感 |
| 空间 | 负空间与正空间的对比 |
| 方向 | 元素方向引导视线 |
| 纹理 | 表面质感的对比 |

### 5.2 构图方法

| 方法 | 特点 |
|------|------|
| 中心对称 | 元素放中心，稳定 |
| 三分法 | 元素放交叉点，动态 |
| 黄金矩形 | 1:1.618比例，和谐 |

### 5.3 裁剪与层级

- 裁剪改变视觉焦点
- 裁剪影响情感表达
- 元素大小、位置决定层级

### 5.4 构图的叙事性

- 物体之间的关系可以讲述故事
- 引入人脸时，观众本能看向眼睛
- 构图不仅是形式，更是意义的传达

## 设计流程

### 标准流程（5步）

```
1. 明确目的
   → 这个设计要传达什么？
   → 目标受众是谁？

2. 选择元素
   → 需要什么图像？
   → 用什么字体？
   → 什么颜色？

3. 构图规划
   → 用什么构图方法？
   → 焦点在哪里？

4. 排版调整
   → 对齐方式
   → 行距行距
   → 字体层级

5. 迭代优化
   → 检查对比
   → 检查可读性
   → 微调细节
```

## 调用示例

### 示例1：海报设计

**用户需求**：帮我做个音乐节海报

**回应流程**：
```
1. 确认信息
   → 音乐节名称和日期？
   → 目标受众（年轻人/专业人士）？
   → 想要的感觉（摇滚/电子/民谣）？

2. 选择元素
   → 图像：音乐相关插图或抽象图形
   → 文字：标题用大字号突出艺人名
   → 颜色：根据音乐风格选择

3. 构图
   → 三分法：标题放上方，艺人名放交叉点
   → 保留足够负空间

4. 排版
   → 标题用粗体大字
   → 时间地点用较小字号
   → 确保可读性
```

### 示例2：传单设计

**用户需求**：帮我做个餐厅传单

**核心要点**：
- 主图：菜品照片或餐厅 logo
- 文字：店名、地址、优惠信息
- 颜色：暖色调（食欲诱发）
- 构图：重点信息突出

## 模块六：图像制作扩展

### 6.1 图像的双重功能

| 功能 | 定义 | 示例 |
|------|------|------|
| 传达信息 | 清晰传递事实或数据 | 产品照片、数据图表 |
| 激发情感与观念 | 传达深层含义 | 艺术海报、宣传画 |

### 6.2 图像的内涵与外延

- **内涵 (Denotation)**：直接意义，即图像字面上表现的内容
- **外延 (Connotation)**：基于文化、情感的联想意义

### 6.3 图像的制作方法

**手工创作**：
- 毛笔、墨水、铅笔、水彩等绘画
- 盲画轮廓、表现性绘画
- 拼贴、刺绣

**摄影与数字**：
- 拍摄、扫描、复印
- Photoshop、Illustrator等软件
- AI获取灵感（参考后重新绘制）

### 6.4 线条的表现力

线条的不同特质传达不同含义：
- **紧密度**：严谨 vs 自由
- **方向**：运动感和导向
- **质感**：粗糙vs光滑、松散vs紧密

## 模块七：排版扩展

### 7.1 字体度量系统

| 单位 | 定义 |
|------|------|
| 点 (pt) | 1pt = 1/72英寸 |
| 派卡 (pica) | 1pc = 12pt |
| em | 相当于字体大小 |
| en | em的一半 |

### 7.2 字体大小选用

| 用途 | 字号 |
|------|------|
| 大标题 | 72-120pt |
| 子标题 | 24-36pt |
| 正文 | 9-12pt |
| 脚注 | 6-8pt |

### 7.3 行距与行长

- **标准行距**：字体大小的1.2-1.5倍（9pt字配12pt行距）
- **最佳行长**：60-70字符（约9-12个单词）

### 7.4 视觉层次结构

通过以下方式创建层次：
- 字体大小差异
- 字体重量（粗体vs regular）
- 颜色深浅
- 行距调整
- 字体类型对比（衬线vs无衬线）

## 模块八：构图扩展

### 8.1 尺度与空间

**尺度 (Scale)**：
- 通过元素大小对比创造层次感
- 大元素不一定最重要，小元素有时更突出

**空间 (Space)**：
- 前景与背景的关系
- 元素的Overlap创造深度感
- 白色空间产生动态张力

### 8.2 构图的叙事性

通过元素关系讲述故事：
- 元素之间的位置关系
- 荒诞组合产生新含义
- 引入人脸时观众本能看向眼睛

## 常见问题

### Q：如何选择字体搭配？
- 限制在2-3种字体
- 选择风格差异大的字体（衬线+无衬线）
- 寻找相似比例或历史时期的字体

### Q：如何让设计更有创意？
- 从虚构客户开始，打破固有框架
- 用Mood Board收集视觉研究
- 尝试不同媒介和技法

### Q：图像如何传达深层含义？
- 用象征和隐喻
- 考虑目标受众的文化背景
- 利用图像与文字的组合

## 模块九：设计历史与流派

### 9.1 包豪斯学派 (1919-1933)

**核心理念**：
- 简洁、现代、抽象形式
- 功能性与美学结合
- 为工业化生产服务

**影响领域**：陶艺、纺织品、电影、建筑、图形设计

### 9.2 瑞士客观设计 (1950s)

**核心理念**：
- 简洁排版，使用无衬线字体 (Helvetica)
- 摄影优于插图，传达事实
- 网格系统组织页面
- 设计作为理性沟通工具

### 9.3 国际主义设计

**代表人物**：约瑟夫·穆勒-布罗克曼、保罗·兰德

**核心理念**：
- 抽象符号与企业形象
- 几何网格统一视觉
- 国际化、无文化偏见的设计语言

### 9.4 设计大师风格

**保罗·兰德**：
- 简洁几何形状传递复杂含义
- 品牌个性化设计
- UPS/ Westinghouse 标志设计

**米尔顿·格拉塞**：
- Big Nudes：简洁框架表达概念
- Bob Dylan 专辑：历史与迷幻结合
- I Love NY：极简情感符号

## 核心原则总结

1. **设计是组合**：四大元素（图像、文字、形状、颜色）组合成设计
2. **对比是动力**：没有对比，设计就平淡无奇
3. **过程大于结果**：创作要迭代，不要追求一次完美
4. **形式与意义**：构图既要美观，也要传达意义
5. **可读性优先**：在表现之前，首先要能读
6. **简洁传递力量**：用最少的元素传达最多的含义
7. **历史启发创新**：学习经典但不死守
