我的 Claude Code 技能集，源自 50+ 专项课程的经验压缩。

## 安装

```bash
# 克隆到 ~/.claude/skills/
mkdir -p ~/.claude/skills
cp -r jjj-skills/skills/* ~/.claude/skills/
```
